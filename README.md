# Sales Report — PT Sejahtera Bersama
**Bank Muamalat Virtual Internship Experience — Business Intelligence Analyst**

## Overview
This project involves building an end-to-end BI solution for PT Sejahtera Bersama by integrating four separate sales datasets into a unified master table and interactive dashboard to support data-driven business decisions.

## Tools Used
- **Google BigQuery** — data import, SQL querying, and table integration
- **Looker Studio** — interactive dashboard and data visualization
- **Google Sheets** — data review and file conversion
- **Draw.io** — Entity Relationship Diagram (ERD)

## Dataset
Four relational tables:
| Table | Rows | Primary Key |
|---|---|---|
| Customers | 2,123 | CustomerID |
| Orders | 3,339 | OrderID |
| Products | 70 | ProdNumber |
| ProductCategory | 7 | CategoryID |


## Key Results
- **Total Sales Revenue:** $1,754,751
- **Total Quantity Sold:** 11,654 units
- **Total Transactions:** 3,339
- **Top Category by Revenue:** Robots ($743,505)
- **Top City by Revenue:** Washington
